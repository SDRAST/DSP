SignalGenerator¶

class DSP.SignalGenerator(mode='analytic', M=1024, components=None, rms=None)¶

Bases: object

Simulated signal generator for making test signals

Method signal() returns a DigitalSignal object with the specified properties. The attribute S has a complex signal.

If components are specified, it is a Fourier series. If not, it is random Gaussian noise.

components - (list of lists) of Fourier components m - (int) data indices mode - (str) “real”, “analytic”, or “complex” S - (DigitalSignal) dtype=complex ndarray subclass
